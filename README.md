# NewsFetch-CLI-App
In this project, students will create a CLI tool that fetches news articles from an external API like NewsAPI. Users can run commands to view the latest news, search articles by keyword, or filter by category and date. The application will store fetched articles in a database and ensure duplicates are not saved.
